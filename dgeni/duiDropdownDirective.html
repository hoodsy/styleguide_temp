duiDropdownDirective

Description

An input field that accepts a search string and renders a list of selectable options accordingly.

Parameter Type Description
getOptions {function} returns a promise which resolves to the options array
model {object} the object that persists the selected option
label {string} the property name of the option object that will be displayed
placeholder {string} the placeholder text of the dropdown
icon {string} the icon that will appear next to the dropdown
